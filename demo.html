Taobao Order Data Extractor Demo

Instructions

This demo shows how to extract and filter order data from Taobao's "已买到的宝贝" (Items I've Bought) page.

  1. Open the Taobao order page in your browser
  2. Open the browser's Developer Tools (F12)
  3. Go to the Console tab
  4. Copy and paste the JavaScript code from order_extractor.js
  5. Use the extractor functions to analyze your order data
